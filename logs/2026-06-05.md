# Working Log — June 5, 2026

## homepage/tutorial.html

Built out the tutorial page from scratch, starting from a bare `<img>` tag pointing at `home.svg`.

### Responsiveness
- Reset `margin`/`padding` on all elements; set `html, body` to `height: 100dvh; overflow: hidden` to eliminate scroll on all screen sizes.
- Centered the image using a flex container (`justify-content: center; align-items: center`) with `padding: 5dvh 5vw` for breathing room on all four edges.
- Image uses `max-width: 100%; max-height: 100%; width: auto; height: auto` so it scales down to fit whichever axis is the constraint while preserving aspect ratio.

### Overlay button positioning
Introduced `.image-wrapper` — a `position: relative` div sized to exactly match the rendered image using:
```css
width: min(90vw, calc(90dvh * 873.41 / 2036.96));
aspect-ratio: 873.41 / 2036.96;
```
This means all `position: absolute` children can use percentage coordinates that map directly to `home.svg`'s viewBox (`0 0 873.41 2036.96`), and those coordinates stay correct at every screen size.
