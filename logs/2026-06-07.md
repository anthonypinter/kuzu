## Layout Structure

```
.game-wrapper
  <header>           ŌåÉ cream bar, centered over board, h1 + attempt counter
  .page-body         ŌåÉ bordered container (dark-panel border, rounded corners)
    .game-area       ŌåÉ board + hint + controls row
    .path-section    ŌåÉ bottom.svg overlay with stat text and live counters
```

### Key layout decisions
- `header` has `max-width: calc(4 * tile-size + 3 * gap-size + 2rem)` to align with the board
- `.game-wrapper` uses `align-items: center` (no max-width) to center the header
- `.page-body` has `border: 2px solid var(--dark-panel)` and `border-radius: 0.75rem`
- `gap: 0` between `.game-area` and `.path-section` so the bottom SVG sits flush

## `time.js` Patches

### 1. Attempt counter format
```js
document.getElementById('attemptCounter').textContent = `ATTEMPT ${this.tryCount}`;
```

### 2. Live counter updates (in `updateGameState`)
```js
const tilesEl = document.getElementById('liveTilesCount');
const powersEl = document.getElementById('livePowersCount');
if (tilesEl) tilesEl.textContent = String(this.victoryStats.tilesRevealed).padStart(2, '0');
if (powersEl) powersEl.textContent = String(this.victoryStats.powersUsed.size).padStart(2, '0');
```

### 3. Null guards for removed warpBubble / grappleBubble
```js
const warpEl = document.getElementById('warpBubble');
if (warpEl) warpEl.classList.toggle('hidden', !this.isWarping);
const grappleEl = document.getElementById('grappleBubble');
if (grappleEl) grappleEl.classList.toggle('hidden', !this.isGrappling);
```
(Same null guard pattern applied in the victory handler.)

### 4. Path stat line population (in `initializeBoard`)
```js
const pathEl = document.getElementById('pathStatText');
if (pathEl && this.solutionData) {
    pathEl.textContent = `${this.solutionData.tiles} TILES USING ${this.solutionData.powers} POWERS`;
}
```

### 5. helpBtn null guard
```js
const helpBtn = document.getElementById('helpBtn');
if (helpBtn) helpBtn.onclick = () => { this.toggleHelpModal(); };
```

### 6. `setupStatsButton` gutted
Stats button no longer injected into header ŌĆö method now only removes any pre-existing button.
